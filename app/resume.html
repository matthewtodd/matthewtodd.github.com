RoleModel Software

January 2005—September 2006

GRITS Inventory Application, A Large Pharmaceutical Company

This application tracks inventories of various biological reagents in freezers around the world, along with their quality assurance histories. We had particular fun writing a Javascript employee-lookup widget backed by the company's LDAP directory and somehow managed to move through 2 ORMs and 3 web frameworks along the way. Written in Java, eventually with WebWork, Hibernate, and PicoContainer.

Assay Annotation Application, A Large Pharmaceutical Company

This is a simple note-taking application on the histories of various experiments. Written in Java with WebWork, Hibernate, and PicoContainer.

Plate Tools Application, A Large Pharmaceutical Company

This application calculates serial dilutions of solutions in arrays of wells, and then reflects their updated states in various legacy databases. Written in Java with OpenSails, a Rails-inspired framework that made judicious use of Java's static typing.

Employee Rewards Application, A Small Startup

This simple rewards catalog, customized for each customer, allowed their employees to search for and select products by tag. Written in Rails, back in the 1.x days.

Web Commerce Group

June 2002—October 2004

Internal Development Server

I donated my old Pentium 133 from college and set it up to host bugzilla, a wiki, our maven repository, project javadocs, a yum repository mirror, and our subversion repositories. Along the way, I also helped the other developers transition to using an early release of Fedora on their desktops.

Arkdom Plus Mobile Device Characteristics

This obfuscated Java library provided mobile device information—screen size, color depth—based on User Agent headers. Far too much of our work involved searching out these parameters on the internet!

Arkdom Commerce Platform

This web application, customized per client, aggregated affiliate marketing links to products around the internet. I wrote tools for parallel Lucene search indexing, set up a failover hosting environment, and eventually earned permission to migrate the codebase away from EJBs.

Rental Agreements, A Large Car Rental Company

As business waned, I was hired out to an experimental technologies group at a large rental car company, flying up to Long Island most weeks. I wrote SOAP request handlers that generated rental agreement forms and sent them to the printers. My work was one of the first projects to emerge from the group, and I received surprisingly enthusiastic feedback on my code.
