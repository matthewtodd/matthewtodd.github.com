Vegetable Ordering Application, Masifio Estates

Live at masifio.co.tz since August 2010, this application receives about 50 orders each week of produce to be shipped from a farm in southern Tanzania to up-market hotels and safari lodges. It receives customer and product lists from Quickbooks and generates packing lists and invoices in return. As internet access is frustratingly slow, I've paid close attention to load times, keeping pages light and assets aggressively cached. Written in Rails, hosted on Heroku.

Dial-up Mail Server, Health Care Technical Services

To generate supplemental income, HCTS provide dial-up email access. I replaced their aging server with a Chef-configured Ubuntu system, providing a web interface for adding and removing customers.

Holiday Card Fundraiser, Amani Children's Home

Seasonally live at cards.amanikids.org each December, this site receives orders for greeting and gift cards, tracking inventory for multiple distributors and accepting payments by PayPal and JustGiving. Over the past 2 seasons, this app has handled hundreds of orders, representing over $20,000 raised for the Amani children. Written in Rails, hosted on Heroku.

Children's Database, Amani Children's Home

This application helps Amani's social workers manage their caseloads and schedule their visits to children's families in the hopes of reunification. From their workflows, it also displays overview statistics for management staff. Written in Rails, originally hosted internally, recently moved to Heroku and S3.

Website, Amani Children's Home

Live at amanikids.org since November 2008, replacing a 5-year-old PHP site. Written in Rails, originally hosted on a Joyent Accelerator, since moved to Heroku and S3.

Internal Server, Amani Children's Home

This Chef-configured Ubuntu system hosted samba shares and the children's database up until its replacement in late 2010 by Dropbox and Heroku. It also provided a number of iptables restrictions in an attempt to reduce Amani's satellite internet charges.

HeadsUp OS X Application

HeadsUp is a simple clone of GeekTool I wrote to explore OS X development — and because I found GeekTool's configuration far too distracting. The first version was in RubyCocoa; I've since rewritten it in Objective-C. I love HeadsUp for keeping my agenda on my desktop, but I can't seem to convince anyone else that it's awesome!

Deploying a Chef Server on Heroku

I dug into Chef's server code just enough to learn how to Rack it up directly. Along the way, I also wrote a minimal CouchDB clone in Rails with a simple JavaScript interpreter. Unfortunately, Chef clients are pretty HTTP-chatty, so using a remote Chef server doesn't yet make much sense in Tanzania.

Captain Ruby Gem

Captain is great for building custom Debian & Ubuntu installation CDs on OS X.

Shoe Ruby Gem

Shoe DRYs up your Rake task configuration without the conceptual weight of Hoe and Jeweler.

Perquackey Ruby Gem

Perquackey displays all the words that can be spelled with a given set of letters. It uses a C extension to avoid creating thousands of throwaway strings as it loops through the dictionary. Try it out at perquackey.matthewtodd.org.

RoleModel Software, January 2005—September 2006

GRITS Inventory Application, A Large Pharmaceutical Company

This application tracks inventories of various biological reagents in freezers around the world, along with their quality assurance histories. We had particular fun writing a Javascript employee-lookup widget backed by the company's LDAP directory and somehow managed to move through 2 ORMs and 3 web frameworks along the way. Written in Java, eventually with WebWork, Hibernate, and PicoContainer.

Assay Annotation Application, A Large Pharmaceutical Company

This is a simple note-taking application on the histories of various experiments. Written in Java with WebWork, Hibernate, and PicoContainer.

Plate Tools Application, A Large Pharmaceutical Company

This application calculates serial dilutions of solutions in arrays of wells, and then reflects their updated states in various legacy databases. Written in Java with OpenSails, a Rails-inspired framework that made judicious use of Java's static typing.

Employee Rewards Application, A Small Startup

This simple rewards catalog, customized for each customer, allowed their employees to search for and select products by tag. Written in Rails, back in the 1.x days.

Web Commerce Group, June 2002—October 2004

Arkdom Plus Mobile Device Characteristics

Arkdom Commerce Platform

Rental Agreements, A Large Car Rental Company

My work was one of the first projects to emerge from the group, and I received surprisingly enthusiastic feedback on my code.
